# ChemEv Modeling Tutorials

This repo contains the tutorials for ChemEv modeling of spherical systems for Project II of AST1420: Galactic Dynamics.

To run the tutorials, you will need to have Python installed along with the following packages:
- numpy
- pandas
- scipy
- astropy
- tables
- matplotlib
- corner
- pocomc
- multiprocess

I recommend using a virtual environment to manage your Python packages. You can create a virtual environment using `venv`:

```bash
python -m venv chemev_env
source chemev_env/bin/activate
```

and then install these packages using pip:

```bash
pip install numpy pandas scipy astropy tables matplotlib corner pocomc multiprocess
```

The main (and only) tutorial notebook is:
- `01_BooI_Example.ipynb`
